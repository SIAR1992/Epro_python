# Epro_python
Denne repository bruges til at dele, modicifere og forbedre koden til denne Python model.

Idéen med Python scriptet er at bearbejde data fra EnergyPro og plotte det på en flot måde, som bliver ensartet imellem alle plots. Dette skyldes at det generalt er meget svært at plotte pænt og ensartet i EnergyPro, samt at den ikke gemmer de modifikationer man har lavet til plottet og meget mere.

Python modellen kræver at man laver nogle installationer til Python, for at kunne kører modellen. Disse opsummeres i word-dokumentet, som benævnes forneden.

I korte træk skal man kopierer produktionsdata fra EnergyPro og smide det ind i et tilfældigt Excel-ark. Man navngiver Excel-arket, som man gerne vil have navnet på plottet skal være. 

Det er vigtigt for nu, at man vælger en prioriteret rækkefølge af produktionsanlæggene enten inde i EnergyPro eller i Excel før man vælger at kører modellen, da det plottes i den prioriteret rækkefølge. 

For at læse fremgangsmetoden mere detaljeret kig gerne på word-dokumentet: "Intro"
